Deflect's JavaScript SDK allows you to integrate bot protection easily into any frontend application including **React**, **Vue**, **Next.js**, or plain JavaScript.

## üì¶ Installation

### ‚úÖ Recommended: CDN (Auto-Updates)

Add the script tag directly to your HTML file:

```html
<script src="https://cdn.jsdelivr.net/npm/@deflectbot/deflect-sdk/dist/index.min.js"></script>
```

This will always serve the latest version of the SDK.

---

### üì¶ NPM (Manual Updates)

If you prefer using a package manager:

```bash
npm install @deflectbot/deflect-sdk
```

Then import it:

```js
import Deflect from "@deflectbot/deflect-sdk";
```

## ‚öôÔ∏è Configuration

Configure the SDK using your `siteKey`:

```js
Deflect.configure({
  actionId: "YOUR_ACTION_ID", // Required
  extraArgs: { mode: "strict" }, // Optional
  forceRefresh: true,           // Optional
});
```

You should call this early in your app's lifecycle (e.g. before making protected API calls).

## üß† Solve a Challenge

Call this when you want to verify a session before taking action (e.g. login, register, payment):

```js
try {
  const token = await Deflect.solveChallenge();
  console.log("‚úÖ Token received:", token);

  // Include this token in your backend API call
} catch (error) {
  console.error("‚ùå Failed to solve challenge:", error);
}
```
