### üì® Request

**Endpoint:**

```
POST https://api.deflect.bot/verify
```

**Headers:**

```http
Content-Type: application/json
```

**Body:**

```json
{
  "api_key": "your-api-key",
  "token": "user-session-token",
  "email": "optional@example.com"
}
```

> üîë `email` is optional, but including it will enrich the response.
